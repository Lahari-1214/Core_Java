#### Creating Your First Java File 
1. <b>Create a New File: </b>
+ Name the file Hello.java. 
+ As we use .js extension for JavaScript and .c for C files, we use .java 
extension for Java files. 
2. <b>Writing the Code: </b>
+ To print "Hello, World!" on the screen as output, we need to write at least 3-4 
lines of code because Java is a structured language. 
+ However, from Java 9, we have JShell, which was introduced for beginners to 
understand the syntax and for experimental purposes. Using JShell, we can 
print this in a single line. 
3. <b>Using JShell: </b>
+ Open JShell by typing jshell in the command prompt. 
+ In VS Code, click on the + icon and select Open Terminal to open the 
terminal.

4. <b>Running the Java Code:</b>
open terminal/cmd then
+ javac filename.java
+ java filename
